== Getting started

. Build project
+
[source,shell]
----
$ mvn package
----
. Run a sample
+
[source,shell]
----
$ java -jar target/lox-1.0-SNAPSHOT.jar samples/hello-world.lox

=== Factorial

.factorial.lox
[source,text]
----
function factorialOf(i) {
    var result = 1;
    while (i > 1) {
        result = result * i;
        i = i - 1;
    }
    return result;
}

print("factorialOf(6) = ");
println(factorialOf(6));
----
